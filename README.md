# Lord of the Rings Brawl

A full stack AJAX JavaScript application for Lord of the Rings fans!

## Technologies Used

- JavaScript
- HTML5
- CSS3
- XML / AJAX
- Leaflet
- [The One API](https://the-one-api.dev/)
